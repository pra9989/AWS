Types of Machine Learnings
There are many types of machine learning and we can group them based on what   problem they solve 
Learning Problems:
 Supervised - using a model to learn a mapping b/w input examples and the target variable
 UnSupervised - using amodel to describe or extract releationships in data
 Reinforcement - an agent operates in an environment and must learn to operate using feedback
 Hybrid Learning Problems:
 Semi Supervised - training data contains very few labeled examples and a large number of unlabeled examples
 self Supervised - Framed as a supervised learning problem in order to apply supervised learning algorthims
 ulti-instance - individual examples are unlabeled; instead bags or groups of samples are labeled
