Components of Generative AI:
Generative AI: the Core components 
Generative AI is not a single entity it s a symphony of interconnected components working together
Understanding these components reveals how these models learn, create and evolve
Made of key components like: Training data model, Architecture, learning Algorithms optimizers and evaluators.
The Foundation: Training Data
The Fuel: Generative AI models learn by ingesting massive amounts of adata releavant to the type of content type will generate.
Diversity is key: The more varied and representative the data the more versatile and capable the model becomes
Data Sources: 
   Text: Books, articles, code websites
   Images: Photographs, painting, drawings
   Audio: Music, speech sound effects
   Video: Movies clips animations

Popular Architecture: 
  Generative Adversarial Networks(GANs)
  Transformer Model
